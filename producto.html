PRODUCTO

Características

  • Diseño: fabricación en madera MDF y guadua que lo hacen eco amigable. Se basa en el concepto DIY (Do It Yourself), que encierra ideas referentes a consumo responsable, reutilización y autogestión.


  • Software: Linux con Raspbian o Ubuntu mate y conoce entornos diferentes a los tradicionales como Windows o IOS.


  • Hardware: El hardware tiene elementos como el procesador, los sensores, la pantalla, los cables y el teclado.

Para los más pequeños, y para aquellos que recién están incursionando en temas de programación, K3OS cuenta con una versión instalada de Scratch, con ejemplos prácticos para fomentar actividades de autoaprendizaje.

Para aquellos que ya manejan con mayor fluidez los aspectos básicos de programación, se tiene software más robusto como lo es processing.

  • Lenguajes de programación: aprende programando por medio de bloques en Scratch o juega Minecraft tambien puedes utilizar python o C/C++


  • Tipo de energía: convencional.
